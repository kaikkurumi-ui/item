# 1.产品V1.2 APP端

#### baseUrl:  http://localhost:8080

 &nbsp;

### 1.1商品列表

 &nbsp;

#### /item/list

 &nbsp;

##### 参数：

##### page: [Int]

##### keyword: [String],(required=false)

&nbsp;

##### 返回：

```json
{
    list: [
        {
            itemImage: [String],
            title: [String],
            price: [Float]
        }
    ],
    isEnd: [Boolean]
}
```

&nbsp;

### 1.2商品详情

&nbsp;

#### /item/info

&nbsp;

##### 参数:

##### itemId: [Long]

&nbsp;

##### 返回:

```json
{
    itemImages: [String[]],
    title: [String],
    price: [Float],
    description: [String]
}
```

# 2.产品V1.2 console端

#### baseUrl: [http://localhost:8081](http://localhost:8080)

 &nbsp;

### 2.1新增商品

&nbsp;

##### /item/create

&nbsp;

##### 参数：

itemImages: [String]
title: [String]
price: [Float]
description: [String]

&nbsp;

##### 返回：

&nbsp;

自增id

**&nbsp;**

### 2.2修改商品

&nbsp;

##### /item/update

&nbsp;

##### 参数:

itemId: [Long]

itemImages: [String]

title: [String]

price: [Float]

description: [String]

&nbsp;

##### 返回：

&nbsp;

成功or失败

&nbsp;

### 2.3删除商品

&nbsp;

##### /item/delete

&nbsp;

##### 参数：

itemId: [Long]

&nbsp;

##### 返回：

&nbsp;

&nbsp;
